# 🛍️ Print India – E-commerce Website

Print India is a modern and responsive **E-commerce Website** built using **HTML, CSS, and JavaScript**.  
This project is designed to showcase and sell printing-related products with a clean, user-friendly interface.

The website provides a smooth shopping experience where users can explore products, learn about the company, and contact for orders or inquiries.

---

## 🌟 Features

- 🏠 **Home Page (Index)** with featured products and highlights  
- 📖 **About Page** describing the business and services  
- 🛒 **Products Page** to display available items  
- 📞 **Contact Page** for customer inquiries and support  
- 🎨 Clean and modern e-commerce UI design  
- ⚡ Smooth animations and hover effects  
- 📱 Fully responsive design (Mobile + Desktop)  
- 🧭 Easy navigation menu  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and dynamic behavior  
- **Font Awesome** – Icons  
- **Responsive Web Design**
